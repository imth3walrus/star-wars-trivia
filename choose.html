Game Rules
  • Designed for 2 users
  • Galactic Empire starts all rounds
  • There is the same amount of questions for both users on every round
  • Difficulty levels: Easy, Medium and Hard
  • Each correct answer is worth 1000 points
  • Time limit of 20 seconds per question
  • If timer gets to 0, question will be skipped.
  • Score Board: Upper left hand side
  • User turn board: Upper right hand side
